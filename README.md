# CampusTalk
<p>A text-based social media platform for college students to <br> share campus experiences, class struggles, funny moments, and study tips.</p>

<h2>Tech Stack</h2>
<p><b>Frontend:</b> Vue 3, TS, Tailwind Css v4.1 <br>
<b>Backend:</b> Node JS, Express <br>
<b>Database:</b> Postgre SQL</p>

<h2>Features</h2>

- Text posts
- Reactions
- Comment threads
- Hashtags
- Trending topics
